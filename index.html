Every generation, lost in folders and downloads.

Most tools treat each render as an isolated output. You type a prompt, get an image, download it — and lose the context of how it was made. Which prompt? Which model? Which campaign? The trail disappears the moment you hit save.

The Constellation System

Not a file browser. A creative observatory.

Every prompt, render, model, style, reference, character, caption, and export becomes a node. Every relationship becomes a visible connection. A character sits at the center like an anchor star — outfits, poses, scenes, videos, and posts orbiting around it, now rendered in a fluid volumetric 3D graph.

  • See where an idea started and how it evolved
  • Know which model produced the best version of a character
  • Spot which looks are drifting from the original identity
  • Surface unused renders that deserve another look
